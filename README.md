# cli_task

Pythonで作成した、CLIベースのタスク管理アプリです。  
コマンドライン上でタスクの追加・表示・完了・削除などの操作を行えます。

## 参考
本アプリは以下の記事を参考にして作成しました。  
ただし、学習目的に合わせてデータ構造や機能を拡張しています。

https://sho-sugimoto.blog/create_cli_application/

## 参考記事からの変更点・工夫点

### データ保存形式を JSON に変更
タスクをテキストではなく JSON 形式で保存しています。  
タスクを辞書構造で管理することで、将来的な拡張
（期限・優先度など）を行いやすい構成にしています。

### タスクに ID を持たせた設計
各タスクには一意な ID を付与しています。  
現時点では削除処理は表示番号（index）を用いて行っていますが、
内部データに ID を持たせているため、将来的に
ID指定操作へ拡張しやすい構成にしています。

### 削除機能を拡張
タスクの削除について、以下の操作を実装しています。

- 表示番号（index）指定によるタスク削除
- 複数番号指定による一括削除
- 全タスク削除

複数削除時には、削除順によるズレを防ぐため、
逆順で削除するようにしています。

### 処理と保存の責務を分離
タスク操作のロジックと、ファイル入出力処理を分けて実装し、
CLIアプリでも責務分離を意識した構成にしています。

## 機能
- タスクの追加
- タスク一覧の表示
- タスクの完了処理
- タスクの一括削除（複数番号指定）
- タスクの全削除
- データはローカルに保存（JSON）

## 使用技術

- 標準ライブラリ（json など）
