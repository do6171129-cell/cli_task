# cli_task

Pythonで作成した、CLIベースのタスク管理アプリです。  
コマンドライン上でタスクの追加・表示・完了・削除などの操作を行えます。

## 参考
本アプリは以下の記事を参考にして作成しました。  
ただし、写経に留まらないよう、設計や機能を拡張しています。

https://sho-sugimoto.blog/create_cli_application/

## 参考記事からの変更点・工夫点

### データ保存形式を JSON に変更
タスクをテキストではなく JSON 形式で保存しています。  
これにより、タスクを辞書構造で管理でき、将来的な拡張
（期限・優先度など）を行いやすくしています。

### タスクに ID を持たせた
表示用の番号ではなく、一意な ID を各タスクに付与しています。  
完了タスクの非表示や削除を行っても、内部データと表示のズレが
起きないようにするためです。

### 「完了」と「削除」を分けて設計
タスクは削除せずに完了状態（`done`）を持たせる設計としています。  
必要に応じて、ID指定による削除や全削除も可能です。

### 削除機能を拡張
以下の削除操作を実装しています。
- ID指定によるタスク削除
- 複数タスクの一括削除
- 全タスク削除

### 処理と保存の責務を分離
タスク操作のロジックと、ファイル入出力処理を分けて実装しています。  
CLIアプリでも責務分離を意識した構成にしています。

## 機能
- タスクの追加
- タスク一覧の表示
- タスクの完了処理
- タスクの削除（ID指定）
- タスクの一括削除
- タスクの全削除
- データはローカルに保存（JSON）

- 標準ライブラリ（json など）
