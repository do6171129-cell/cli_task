# cli_task

Pythonで作成した、シンプルなCLIタスク管理アプリです。  
コマンドライン上でタスクの追加・一覧表示・完了処理を行えます。

## 参考
本アプリは以下の記事を参考にして作成しました。  
ただし、写経に留まらないよう、設計や実装の一部を変更しています。

https://sho-sugimoto.blog/create_cli_application/

## 参考記事からの変更点・工夫点

### データ保存形式を JSON に変更
タスクを単純なテキストではなく、JSON形式で保存するようにしました。
これにより、辞書構造でタスクを管理でき、将来的な拡張
（期限・優先度など）を想定しやすくしています。

### タスクに ID を持たせた
リストの index ではなく、一意な ID を各タスクに付与しました。
完了タスクの非表示などを行った際に、表示順と内部データのズレが
起きないようにするためです。

### 「削除」と「完了」を分けて設計
タスクを削除するのではなく、`done` フラグで完了状態を管理しています。
データは保持したまま、表示を制御する設計にしました。

### 処理と保存の責務を分離
タスク操作のロジックと、ファイル入出力の処理を分けて実装しました。
CLIアプリでも、役割ごとにコードを分ける意識を持つためです。

## 機能
- タスクの追加
- タスク一覧の表示
- タスクの完了処理
- データはローカルに保存（JSON）

## 使用技術

- 標準ライブラリ（json など）
